# RAG-Based Diabetes Care Model

This module contains the Retrieval-Augmented Generation (RAG) pipeline
for the Chronic Disease Care Planner project.

### Features
- FAISS-based semantic search over ADA guidelines
- Sentence-Transformers embeddings
- Structured guideline extraction
- Personalized diabetes care recommendations

### How to Run
```bash
cd rag_based_model
python setup.py
python main.py
